Sn = 1+2+...+n = n(n+1)/2  
$$
Sn = 1 + 2 + ... + n = \frac {n(n+1)}{2}
$$


### 1、使用数学归纳法证明：
1、假如Sn = n(n+1)/2为真、可以推导出第(n+1)项也为真  
也就是说、可以由正确的前一项推导出后一项也正确  
2、第1项是正确的  
那么、我们就可以保证对所有n∈R、Sn = n(n+1)/2都成立  
```
假如Sn = n(n+1)/2为真，那么第(n+1)项为：  
1+2+...+n + (n+1)
= n(n+1)/2 + (n+1)
= (n(n+1) + 2(n+1))/2
= ((n+1)(n+2))/2
```
即第(n+1)项也为真  
所以对所有n ∈ N*、Sn = n(n+1)/2都成立  

### 2、另一种证明方法
```
S1 = 1 + 2 + ....... + n
S2 = n + (n-1) + ... + 1
S1 + S2 = 2S = (n+1)n
=> S = n(n+1)/2
   n ∈ N*
```

## 二、更一般的表述
Sn = S0 + (S0+d) + ... + (S0+nd) = [(n+1)(2S0+nd)]/2  
n ∈ N、d为公差  
这也可用上面的两种方法分别证明
